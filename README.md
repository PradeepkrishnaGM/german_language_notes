# German Notes

A compact, GitHub-friendly set of German grammar and usage notes.

## Overview

These notes cover core German topics including:
- verb conjugation and modal verbs
- noun gender, articles, and possessives
- the four grammatical cases
- prepositions and case changes
- major tenses
- conjunctions, clauses, and sentence structure
- negatives, questions, imperatives, and passive voice
- adverbs, comparison, numbers, dates, and time

## Repository Structure

```text
.
├── README.md           # Intro page and navigation
└── german-notes.md     # Full notes in Markdown
```
